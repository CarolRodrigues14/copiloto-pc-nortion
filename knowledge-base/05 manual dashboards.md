# Manual de Acesso a Dashboards — Nortion Capital

## Ferramentas Utilizadas
- **Power BI:** dashboards executivos e de liderança (performance agregada, turnover, custos de P&C)
- **Metabase:** dashboards operacionais do time de P&C (acompanhamento de rotinas, qualidade de dados)

## Como Solicitar Acesso
1. Abrir chamado no sistema interno de TI, categoria "Acesso a Dashboard"
2. Informar: nome do dashboard, justificativa de necessidade, aprovação do gestor direto
3. TI concede acesso conforme o nível de dados definido na Política de Segurança (ver documento correspondente)
4. Prazo médio de concessão: 2 dias úteis

## Dashboards Disponíveis

### Dashboard de Performance por Time (Power BI)
- Métricas: produtividade, performance individual (agregada por time), curva de aprendizagem média
- Acesso: gestores (nível 2) e time de P&C (nível 3)
- Atualização: mensal, todo dia 5 do mês seguinte

### Dashboard de People Analytics (Power BI)
- Métricas: turnover, custo de contratação, ROI de treinamento, indicadores preditivos
- Acesso: time de P&C (nível 3) e liderança executiva (nível 4)
- Atualização: mensal

### Dashboard Operacional de Qualidade de Dados (Metabase)
- Métricas: percentual de registros validados, duplicidades encontradas, inconsistências de campos
- Acesso: time de P&C (nível 3)
- Atualização: semanal

## Boas Práticas de Uso
- Não exportar dashboards inteiros para fora dos sistemas corporativos sem justificativa registrada
- Sempre verificar a data de última atualização antes de tomar decisões com base nos dados exibidos
- Reportar inconsistências de dados ao time de P&C através do canal interno, não editando o dashboard diretamente
