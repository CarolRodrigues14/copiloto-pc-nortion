# Política de Segurança da Informação e LGPD — Nortion Capital

## Princípios Gerais
A Nortion Capital trata dados pessoais de colaboradores, candidatos e clientes em conformidade com a
Lei Geral de Proteção de Dados (LGPD — Lei 13.709/2018). O tratamento de dados sensíveis segue o princípio
da minimização: coleta-se apenas o necessário para a finalidade declarada.

## Classificação de Dados
- **Dados públicos:** informações institucionais sem restrição (ex: organograma geral)
- **Dados internos:** informações de uso interno, sem caráter pessoal sensível (ex: políticas, processos)
- **Dados pessoais:** informações que identificam uma pessoa (ex: nome, e-mail, cargo)
- **Dados sensíveis:** informações de saúde, dados de desempenho individual detalhado, dados biométricos —
  exigem controle de acesso restrito e justificativa de necessidade

## Níveis de Acesso a Dados de P&C
| Nível | Quem tem acesso | O que pode ver |
|---|---|---|
| Nível 1 | Todos os colaboradores | Políticas públicas, dicionário de métricas geral |
| Nível 2 | Gestores | Performance da própria equipe, dados agregados |
| Nível 3 | Time de P&C | Dados individuais de performance, dados sensíveis de RH |
| Nível 4 | Especialista de Performance + liderança executiva | Dados sensíveis + análises preditivas de People Analytics |

## Regras de Uso de Dados Sensíveis
- Dados individuais de desempenho não devem ser compartilhados fora do time de P&C sem anonimização, exceto
  com o próprio colaborador e seu gestor direto.
- Qualquer exportação de planilha contendo dados sensíveis deve ser registrada (quem exportou, quando, finalidade).
- Ferramentas de IA (como o Copiloto de P&C) que processam dados sensíveis devem operar apenas sobre dados
  já anonimizados ou agregados, salvo quando o usuário autenticado tiver nível de acesso 3 ou 4.

## Segurança da Informação
- Senhas devem seguir política de complexidade mínima e autenticação de dois fatores (2FA) obrigatória para
  sistemas que acessam dados de nível 3 ou 4.
- Incidentes de segurança (vazamento, acesso indevido) devem ser reportados ao DPO (Encarregado de Dados)
  em até 24 horas da identificação.

## Direitos do Titular dos Dados (LGPD)
Colaboradores podem solicitar, a qualquer momento: acesso aos próprios dados, correção de dados incorretos,
e exclusão de dados quando não houver base legal para retenção (respeitando obrigações trabalhistas/fiscais).
